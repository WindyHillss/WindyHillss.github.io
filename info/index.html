Profile Card

This profile card showcases Discord statuses such as active, online, or offline, along with the user's profile picture. The data is fetched in real-time from https://api.lanyard.rest/. The card is designed to resemble a Windows-style window and can be freely dragged around the screen by holding the top title bar.

The bottom area of the card provides space for custom links, with subtle hover effects for better interactivity. Additionally, the card content can be used as a link to navigate directly to the radio site.

To create a visually immersive experience, a leaf animation script harmonizes with the background image, and a campfire sound adds ambiance. The sound only plays after user interaction to comply with browser autoplay policies.

Social media links such as Pinterest, Discord, and Spotify are included within the card, each with hover effects to indicate interactivity.

At the bottom-left of the site, author and copyright information is displayed, while the bottom-right provides attribution for the background image with a clickable link.

The site also includes an Easter Egg: typing a secret keyword ("gwen") triggers a dynamic video background, enhancing the interactivity and surprise element of the site.

Overall, this site is designed to inform users while providing a rich interactive experience combining live data, animations, sound effects, and playful hidden features.

Radio

This interactive radio box allows users to listen to a live streaming radio station directly from the site. The currently playing station is displayed with a clickable link to access it in a new tab.

Users can also download the radio player application directly via the provided download link. The audio will only start playing after user interaction, in compliance with browser autoplay policies.

The radio box includes a volume slider for real-time adjustment and a duration timer that shows how long the user has been listening.

The box mimics a Windows-style window, with draggable functionality. Users can move the radio box anywhere on the screen by holding the title bar at the top.

Additionally, a background leaf animation runs behind the interface, creating a visually immersive experience that complements the site's overall aesthetic.
